## 功能

1. 自动删除B站 **已开奖的转发抽奖** 动态和 **源动态已被删除** 的动态；

2. 自主选择删除转发的 **某个用户动态** 的转发动态；

3. 删除 **X天及之前** 发布的源动态的转发动态；

4. 按 **日期范围** 删除转发动态，支持 **关键词过滤**；

5. 删除动态并 **取关用户**；

6. **自定义请求延迟**，随机间隔降低风控风险；

7. **进度条面板** 实时显示任务执行状态；

8. 任务完成后弹出 **总结弹窗**，需手动关闭，方便事后查看。

## 用法

首先需要安装油猴扩展，然后安装用户脚本。

[点击安装](https://raw.githubusercontent.com/monSteRhhe/bilibili-dynamic-del/main/bili-dynamic-autodel.user.js)

之后可以在B站网站内点击油猴扩展菜单里的按钮使用：

![菜单](./screenshots/menu.webp)

### 删除模式

#### 1. 自动判断

自动删除源动态有互动抽奖且已经开奖的动态和源动态已被删除的转发动态。

#### 2. 指定用户

点击后在弹出窗口输入用户名或者该用户的数字UID并确定，能够删除源动态的作者是该用户的转发动态。

如果需要删除多个用户名/UID，可以使用英文逗号「,」分割。([issue #4](https://github.com/monSteRhhe/bilibili-dynamic-del/issues/4))

#### 3. 删除X天前的转发动态

点击后在弹出窗口输入往前的天数，源动态在这天以及之前的日期发布的转发动态会被删除。([issue #3](https://github.com/monSteRhhe/bilibili-dynamic-del/issues/3))

判断的日期是转发的 **源动态发布的日期** 而不是转发动态的日期，范围是设定的日期及之前的日期。

#### 4. 日期范围+关键词

点击后依次输入起始日期、结束日期和关键词：

- **日期格式**：`YYYY-MM-DD`，如 `2024-01-01`；
- **关键词**：可留空（不限制），多个关键词用英文逗号「,」分隔（匹配任一即删除）；
- **关键词匹配范围**：可在设置中选择「源动态正文」或「转发评论」或「源动态正文+转发评论」；
- **安全机制**：自动排除未开奖的抽奖动态，不会误删进行中的抽奖。

### 进度条面板

执行任务时，页面右下角会显示实时进度条面板，包含：

- 当前状态文字（扫描第N页、查询抽奖、删除动态…）
- 已扫描数 / 已删除数 / 当前页码
- 动画进度条

可在设置中关闭。

### 任务总结弹窗

任务执行完成或异常终止后，会弹出总结弹窗：

- 显示执行模式、扫描数、删除数、失败数、取关数、耗时等
- **必须手动点击「知道了」才会关闭**，避免用户离开后无法查看结果
- 异常终止时显示错误详情

### 设置

点击油猴菜单中的「打开设置」可打开设置窗口：

![settings](./screenshots/settings.webp)

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 启用取关功能 | 关 | 开启后删除动态时会自动取关对应UP主 |
| 显示进度条面板 | 开 | 执行任务时是否显示实时进度浮窗 |
| 弹出系统通知 | 关 | 是否弹出浏览器系统通知 |
| 关键词匹配范围 | 源动态正文+转发评论 | 仅对「日期范围+关键词」模式生效 |
| 请求延迟 最小值 | 1000ms | 每次请求之间的最小随机延迟 |
| 请求延迟 最大值 | 3000ms | 每次请求之间的最大随机延迟 |

## 其他

黑猴子/脚本猫测试都可用。

在使用时要注意功能的适用范围，**<u>数据无价，谨慎使用</u>**：

- 这里的 **转发抽奖动态** 指的是是带有![互动抽奖按钮](./screenshots/lottery.webp)的，可以看到开奖倒计时和中奖用户的"互动抽奖"动态；

  **源动态已被删除的动态** 就是如图所示这种![源动态已删除](./screenshots/deleted.webp)

- 指定用户的删除输入的则是源动态作者的用户名/UID，比如转发的是用户"ABCABC"（uid01234567）的动态，要删除自己转发的ta的动态，要输入的就是"ABCABC"或者"01234567"；

- 按日期删除输入的是往前推的天数，比如今天22号，要删除12号及之前的就输入10。

- 请求延迟默认为 1~3 秒的随机间隔，可在设置中调整。建议不要低于 500ms，否则可能触发B站风控。
